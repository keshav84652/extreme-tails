# My First Post

Welcome to my digital garden! This is my first post using [[index|Quartz]].

## What is Quartz?

Quartz is a static site generator that transforms your markdown files into a beautiful, interactive website.

### Key Benefits

1. **Fast**: Static sites load quickly
2. **Searchable**: Built-in full-text search
3. **Connected**: Automatic backlinks and graph view
4. **Customizable**: Themes and layouts
